1.9

  

A = {1,2}     B = {a,b}

A x B = 

{ (1,a), (1,b) }

 (2, a), (2, b)

  

If A or B is empty

A x B = B x A          A x B = {}

  

A = {1,2}         B = {}

A x B = {}

  

A x B = B x A

If A = B

  

A x B squiggly arrow A x A

A x A = A x A

B x A = {}

  

A subset of A x B is a relation from A to B. A relation to itself is called a relation on A

  

Ex. 

A = {1,2}

B = {a,b,c}

  

A x B = 

{ (1,a) (1,b) }
